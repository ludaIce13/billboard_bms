# Billboard Management System Frontend

A simple React + Tailwind starter for the Billboard Management System (BMS).

Includes:
- Basic Dashboard
- Test page for REVMIS API placeholder

## Usage
1. Navigate to `frontend/`
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the dev server:
   ```bash
   npm run dev
   ```
4. Visit [http://localhost:5173](http://localhost:5173)
